* 安装

  ```shell
  npm install vditor --save
  ```

* 引入

  ```javascript
  import Vditor from 'vditor'
  import "~vditor/src/assets/scss/classic" // 或者使用 dark
  
  const vditor = new Vditor(id, {options...})
  vditor.focus()
  ```
